# Glossary

> SME, define the **clinical terms** Willow may use correctly with
> users (and only those — the bot avoids any term not on this list).
>
> Each entry has the term, a one-line plain-English definition, and
> a sentence about when (and when not) Willow may use it.

## Format

```
### Term
**Plain definition:** [SME: in user-friendly words]
**Use when:** [SME: situations where this term is helpful]
**Don't use when:** [SME: situations where it would be inappropriate]
```
