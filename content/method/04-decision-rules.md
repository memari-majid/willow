# 04 — Decision Rules

> SME, this file is the **"if X then Y" rulebook** Willow follows.
> Plain English — the model reads it directly.
>
> Each rule lives under a short heading so you (and the next reviewer)
> can find it later. Replace every `[SME: …]` with your rules.

## Rule format

Use this shape so the rules stay readable:

```
### Rule — short name
**If:** [the signal in the user's message or the session]
**Then:** [what Willow does, in order]
**Why:** [the clinical or ethical reason — for your future self]
```

> covers: how to respond to acute distress short of crisis, how to
> handle requests for advice you won't give, how to handle
> clinical-sounding questions ("do I have ADHD?"), how to respond to
> a returning user who picks up where they left off, and how to
